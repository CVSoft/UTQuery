RemoteScoreboard
Version 1.0
Written by notipa
=================

Instructions
============
  To select a server, use the server IP address and query (NOT game) port as 
commandline parameters. For example:
RemoteScoreboard.py 74.91.115.188 7778
If a port is not specified, the default is used.

  If the game type is incorrectly shown as a non-team game type, you can force
display as teams with the following syntax:
RemoteScoreboard.py 74.91.115.188 7778 override
